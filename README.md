# Forensic-Image-Download
Download for an E01 image that can be used to practice digital forensic investigation skills. The download includes the image, an Instagram data download, an answer key, a case letter, and two chain of custody forms. 

The main goal of this project was to provide an accessible and useful resource for students, educators, and new investigators who want hands-on experience with a current digital forensic case.

This case is mainly for Axiom, Autopsy, and FTK Imager. Feel free to use any other tools you wish, though the answer key may not be much help besides showing you what evidence to look for.
